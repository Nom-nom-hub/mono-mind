Key Features

  • Repository Analysis - Deep dependency graph analysis and impact assessment
  • Incremental Builds - Smart building that only rebuilds what's necessary
  • Multi-language Support - Works with Go, JavaScript, TypeScript, Python, and more
  • Plugin System - Extensible architecture for custom workflows
  • Visualization - Interactive dependency graphs and reports
  • Release Management - Automated versioning and changelog generation

Installation

# Download the latest release
curl -L -o mono.exe https://github.com/nom-nom-hub/mono-mind/releases/latest/download/mono-windows-amd64.exe

# Or build from source
git clone https://github.com/nom-nom-hub/mono-mind.git
cd mono-mind
go build -o mono.exe ./cmd/mono

Then run:

./mono.exe --help
