Installation

Get started with MonoMind in seconds. Choose your preferred installation method:

Windows

# Download the latest release
curl -L -o mono.exe https://github.com/nom-nom-hub/mono-mind/releases/latest/download/mono-windows-amd64.exe

# Run
./mono.exe --help
Download Windows Binary

macOS

# Download the latest release
curl -L -o mono https://github.com/nom-nom-hub/mono-mind/releases/latest/download/mono-darwin-amd64

# Make executable and run
chmod +x mono
./mono --help
Download macOS Binary

Linux

# Download the latest release
curl -L -o mono https://github.com/nom-nom-hub/mono-mind/releases/latest/download/mono-linux-amd64

# Make executable and run
chmod +x mono
./mono --help
Download Linux Binary

Build from Source

For developers who want to build from source or contribute to MonoMind:

git clone https://github.com/nom-nom-hub/mono-mind.git
cd mono-mind
go build -o mono ./cmd/mono
./mono --help
